# MD CarHud - FREE

- Simple & Minimalist FiveM CarHud using smooth RPM bar and neon colors!
- Fuel level & Engine health idicator
- Highly configurable
- Open Source
________________________________________________________________________________________________________

- Download md_carhud, drop it to your FiveM server resources, add ensure md_carhud to your server.cfg and enjoy!
